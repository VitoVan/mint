# Usage

```bash
cd dashboard
# install dependencies
yarn install
# start the dev server
yarn start
```

- How to use tmux

  ```text
  tmux new -s <session-name> 新建会话
  $ tmux detach 分离会话
  $ tmux ls 查看当前所有的 Tmux 会话
  $ tmux attach -t 0  接入会话
  $ tmux kill-session -t 0 命令用于杀死某个会话。
  $ tmux switch -t 0 切换会话
  ctrl+b [ 滚屏
  Ctrl+b d 分离会话
  Ctrl+b ? 帮助
  Ctrl+b %
  Ctrl+b "
  Ctrl+b up
  Ctrl+b down
  Ctrl+b left
  Ctrl+b right
  Ctrl+b d：分离当前会话。
  Ctrl+b s：列出所有会话。
  Ctrl+b $：重命名当前会话。
  ```
